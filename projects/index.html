Some of my closed source projects

Smapps – Find Similar Android Apps - 2016

  • Flask based Web application hosted on heroku with MongoDB and redis.
  • Developed a backend system which groups similar android apps by their functionality with the help of clustering algorithms.
  • User can view apps similar to his/her favourite apps.
  • Custom Flask based API is designed to connect backend system with Web application.

Quartz - Web Crawler & Scraper - 2015

  • Crawls shopping portals like amazon, flipkart to find available products & scraps detail about the product.
  • Python based crawler & scraper with MongoDB as database hosted on heroku.

Foodle – Real Time Restaurant Table Reservation Service - 2015

  • Developed with Meteor Framework and hosted on Meteor’s own hosting platform.
  • Shows availability of table in restaurant along with restaurant details.

Orbit – Contact Sync Android App - 2014

  • Syncs user’s contact details with all the contacts present in its address book.
  • Developed an API service in PHP with MySQL as database which syncs contacts in regular intervals.
  • Developed an android app as UI with SQLite as local database.
