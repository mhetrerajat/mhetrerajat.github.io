Backpacker

June 2016

Backpacker is a smart travel guide, which will help you plan your trips the best way possible. Personalise your trip by exploring tourist attractions, hotels, restaurants, stores and other amazing places. Backpacker will show you detailed information along with real reviews about each and every place you want to visit.

  • Backpacker is an android app developed as capstone project for android developer nano degree program.
  • Google Places API is used to fetch information about places nearer to user’s desired location(i.e city).
  • Realm is used as local db to store data which is accessed from network.
  • Libraries like eventbus, picasso, gson is used for communication between app components, image caching and serialising & deserialising JSON respectively.
