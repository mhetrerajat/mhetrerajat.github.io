# My Personal Portfolio

This repository contains the source code for my personal portfolio website, hosted at [mhetrerajat.github.io](https://mhetrerajat.github.io).


## How to Use/Run Locally

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/mhetrerajat/mhetrerajat.github.io.git
    ```
2.  **Navigate to the project directory and set up the development environment:**
    ```bash
    cd mhetrerajat.github.io && pnpm install && pnpm run dev
    ```
3.  **Open the provided local address in your web browser.**
